# Security Policy

## Supported Versions

Security fixes are provided for the latest published data tag only.

This repository does not maintain LTS branches or backport security fixes to older profile-data releases. The `main` branch is a development branch; fixes may land there before a tag, but downstream implementation repositories should import fixed tags for releases.

| Version / channel | Security fixes |
| --- | --- |
| Latest published tag | Yes |
| Older tags | No |
| LTS / backport branches | No |

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Report vulnerabilities privately via [GitHub Security Advisories](https://github.com/fa-yoshinobu/plc-comm-slmp-profiles/security/advisories/new) or by contacting the maintainer directly.

Include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Affected profile data or generated JSON file
- Suggested fix, if available

You can expect an acknowledgment within 7 days.
